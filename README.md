# Portfolio

Welcome to my personal portfolio website! This site showcases my projects, skills, experience, and provides a way to contact me. Built with **React** and **Tailwind CSS**, it features a responsive design and smooth scrolling for an interactive user experience.

** View my website here: ** [katherinehutchinson.com]{katherinehutchinson.com}
---

## Features

- **Home Section:** Introduction and brief overview of who I am.
- **Experience Section:** Details about my internships, hackathon leadership, and other professional experience.
- **Skills Section:** Interactive skill cards with icons for programming languages, frameworks, and tools.
- **Projects Section:** Portfolio projects with hover effects and pop-up modals for details and demos.
- **Contact Section:** Clickable email, LinkedIn, and GitHub links for easy communication.
- **Responsive Design:** Works seamlessly on mobile, tablet, and desktop devices.
- **Smooth Scroll Navigation:** Navigation bar links scroll to the respective sections of the page.

---

## Technologies Used

- **React** – For building reusable components and managing UI state.
- **Tailwind CSS** – For styling and responsive layouts.
- **React Icons** – For skill icons and social media links.
- **React Scroll** – For smooth scrolling navigation between sections.
